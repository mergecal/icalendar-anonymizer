# Contributing to icalendar-anonymizer

Thanks for contributing!

Full contributing guide: https://icalendar-anonymizer.readthedocs.io/stable/contributing.html

## Quick Start

Fork and clone the repository:

```bash
git clone https://github.com/mergecal/icalendar-anonymizer.git
cd icalendar-anonymizer
```

Install with development dependencies:

```bash
pip install -e ".[dev]"
```

Run tests:

```bash
pytest
```

Follow [Conventional Commits](https://www.conventionalcommits.org/) format for commit messages.
